Softbox Studios is a Trinidad photography and media production company. This was an early client project — one of the first commercial websites built from scratch using hand-coded HTML, CSS, and JavaScript.

## Context

This predates WordPress becoming the default for client work. Everything was written by hand: the layout, the navigation, the gallery, the contact form. It was a useful exercise in doing things without a framework, and it held up well for the client.

## What was built

- Static HTML/CSS/JavaScript site
- Photography portfolio gallery
- Services and contact sections
- Cross-browser compatible layout for the time
