RL Photography is the portfolio site for Rainier Lange, a Trinidad-based photographer. The goal was to let the photography speak — clean presentation, fast loading, and nothing in the way.

## What was built

Built on WordPress with a modified theme optimised for photography portfolios. The layout prioritises large imagery and a minimal interface so the work gets the attention it deserves. Portfolio galleries are organised by category, making it easy for prospective clients to find relevant work quickly.

## Key features

- Gallery-focused WordPress layout
- Modified theme for photography presentation
- Category-organised portfolio sections
- Mobile-responsive design

The site is still live at [rainierlangephotography.com](https://rainierlangephotography.com/).
