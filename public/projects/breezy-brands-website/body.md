Breezy Brands is a Trinidad-based business. This project covered both the interface design and the final WordPress build — one cohesive engagement from concept through to launch.

## Process

Started in Adobe XD with wireframes and a design direction agreed on with the client before any code was written. The design was then translated into a custom WordPress theme, keeping the final product faithful to what was approved.

## What was built

- UI/UX design in Adobe XD
- Custom WordPress theme development
- Responsive layout for desktop and mobile
- Client-ready CMS so the team could manage content themselves after handoff
