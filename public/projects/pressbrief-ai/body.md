## Overview

PressBrief AI is a WordPress plugin that integrates six AI providers (OpenAI, Claude, Gemini, Groq, DeepSeek, Perplexity) to generate structured summaries for any post type. The free version is wp.org-compliant and fully standalone; the Pro add-on extends it with bulk processing, white-label branding, and a license management system.

## Highlights

- Six AI provider integrations — OpenAI, Claude, Gemini, Groq, DeepSeek, and Perplexity — switchable per site
- Free version is fully standalone and WordPress.org-compliant with no external dependencies
- Pro add-on adds bulk processing, white-label branding, and a license key system
- Accompanying Next.js site doubles as a marketing page and license API server with activation/deactivation endpoints
- SQLite-backed admin dashboard designed to migrate to PostgreSQL for production scale
