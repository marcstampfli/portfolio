## Overview

Built to bring the traditional Caribbean card game All Fours to the web, this project recreates the social and competitive feel of the game in a modern online experience. Players can join matches, track performance, view profiles and leaderboards, and replay completed games through a polished real-time interface.

## Highlights

- Custom game logic handling the full rules and scoring of All Fours
- Real-time multiplayer state management built with Supabase and Zustand
- Full product layer around the core game: profiles, leaderboards, and game replays
- Test coverage with Jest (unit) and Playwright (end-to-end)
