## Overview

All Fours Ultimate is a full-stack esports platform built around the Trinidad & Tobago national card game, designed to bring a deeply cultural pastime online with the polish of a modern competitive title. The frontend is a React 19 + Vite SPA written in strict TypeScript, paired with a contract-first Django REST API backend. The platform is deployed across Vercel (frontend), Railway (API), and Cloudflare (DNS, CDN, SSL) at allfours.io.

## Highlights

- Elo-based ranked matchmaking and single-elimination tournaments
- Real-time WebSocket gameplay with spectator mode and a replay viewer
- Leaderboards and in-game chat for the competitive community
- Practice mode powered by a custom bot AI engine
- Fully custom "Caribbean Midnight Esports" CSS design system — bioluminescent teal, gold, and coral accents over a deep ocean palette
