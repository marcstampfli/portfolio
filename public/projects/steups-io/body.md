steups.io is a solo-built culture platform for Trinidad and Tobago, housing 38 "tings" across three categories: practical social tools (bill splitters, lime planners, outage kits), AI-powered generators (Trini rewriters, clapbacks, excuses, horoscopes), and daily games (Wordle, Connections, price-guessing, bracket pickers).

## What was built

The build features a custom TypeScript engine layer for generation and game logic, Groq LLM integration for culturally-grounded AI content, dynamic OG image generation for social sharing, a Supabase-backed XP and leaderboard system, and seasonal theming tied to the T&T cultural calendar.

A deliberately local product built with global engineering standards.

## Stack

- **Frontend:** Next.js, React, TypeScript, SCSS Modules, Framer Motion, Radix UI
- **AI:** Groq AI (LLM integration for generators and cultural content)
- **Backend/Data:** Supabase (XP system, leaderboards, persistence)
- **Validation:** Zod
- **Testing:** Vitest, Playwright
- **Deployment:** Vercel
